# HUGPONG – Agricultural Management Platform

An integrated management system designed for the **Silay Block Farm** operations. This project bridges the gap between farm owners, supervisors, and field workers through a unified mobile and web ecosystem.

## 📂 Project Structure

- **`/mobile`**: React Native (Expo) application for farm supervisors and workers.
  - Offline-first data synchronization.
  - Market price monitoring (HPCo Silay & Molasses).
  - Operational scheduling and task management.
  - Income and yield calculator.
  
- **`/admin`**: Web-based administration console for farm owners.
  - Real-time dashboard and KPI monitoring.
  - Centralized price management.
  - Detailed reports and analytics.
  - Personnel and plot management.

## 🚀 Getting Started

### Mobile App
1. Navigate to `mobile/`
2. Install dependencies: `npm install`
3. Start the project: `npx expo start`

### Admin Portal
1. Navigate to `admin/`
2. Open `index.html` in any modern web browser.

## 🌿 Core Features
- **Real-time Market Sync**: Monitor HPCo and MOL prices.
- **Offline Operations**: Log tasks in the field without internet; sync later.
- **Data-Driven Decisions**: Advanced profitability and revenue calculators.
- **Secure Sync**: Conflict resolution and device audit logs.
